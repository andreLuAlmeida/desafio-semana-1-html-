# Dinossauros!

## Descrição
Este projeto consiste na criação de um blog simples sobre dinossauros utilizando HTML5 semântico e CSS básico. O objetivo foi aplicar os fundamentos apresentados no curso, com foco na estrutura correta da página, organização do conteúdo e clareza do código, sem priorizar design avançado.

A página possui cabeçalho, conteúdo principal com posts, sidebar com categorias e rodapé.

## Tecnologias utilizadas
- HTML5
- CSS3
